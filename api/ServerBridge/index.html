ServerBridge.Fire() → void

Equivalent of RemoteEvent:FireClient(), the ServerBridge:Fire() function takes a player type, or a singular player, and fires the data to the client. Important to note that it takes a singular object as a parameter instead of using tuples- this is intentional.
